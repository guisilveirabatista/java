How HashMap Works:

1. Hashing: When you add a key-value pair, the key's hashCode() is calculated.
2. Index Calculation: The hash is used to determine which bucket (array index) to use: index = hash & (array_length - 1)
3. Collision Handling: If multiple keys hash to the same bucket, they're stored as a linked list (or tree for many collisions).
4. Retrieval: To get a value, the key is hashed again to find the bucket, then the list is searched for the exact key.
